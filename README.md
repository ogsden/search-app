# Search App

Тестовое задание — адаптивное приложение на Angular с поисковой строкой и панелью фильтров.

## Макет

Приложение реализовано для двух разрешений:
- **414px** — мобильная версия
- **1440px** — десктопная версия

## Возможности

- Анимированное раскрытие строки поиска (раздвигается влево при нажатии на иконку)
- Панель фильтров появляется при фокусировке поисковой строки
- Адаптивная вёрстка: мобильная и десктопная версии
- Компонентная архитектура с чётким разделением ответственности
- BEM-методология в CSS
- Angular Signals для управления состоянием
- Angular Animations для плавных переходов

## Стек

- Angular 21
- TypeScript
- SCSS
- Angular Animations

## Структура компонентов

```
app/
├── components/
│   ├── header/          — шапка (мобильная + десктопная)
│   ├── search-bar/      — строка поиска с кнопкой
│   ├── filter-panel/    — фильтры (автор, чекбоксы, секция "Только")
│   └── search-history/  — история поиска (десктоп)
├── app.ts               — корневой компонент
├── app.html             — шаблон с responsive layout
└── app.scss             — стили layout
```

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
ng serve

# Открыть http://localhost:4200
```

## Сборка

```bash
ng build
```

Собранное приложение будет в папке `dist/`.
